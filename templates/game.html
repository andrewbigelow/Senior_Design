Cognitive Overload Training

Welcome! This game challenges your ability to multitask under pressure.

📋 How It Works:

The game has TWO phases:

  • Phase 1 (Rounds 1-3): Complete ONLY visual tasks - click images that match the letter shown
  • Phase 2 (Round 4+): Audio tasks are ADDED - you must listen to spoken numbers/words while completing the visual task, then answer BOTH!
  • Difficulty increases: More images, less time, faster pace
  • Team communication: Say a teammate's name to request help during challenges

🎮 Visual Task Example:

Instruction: "Click on all images that start with the letter B"

🍌 Banana ✓
🚗 Car
🐝 Bee ✓
🌸 Flower

✓ Click items with gold borders (Banana, Bee) = Correct answer!

🎧 Audio Task Example (Round 4+):

What happens: While you're doing the visual task, you'll hear an audio clip with a task like:

"Count all even numbers. 7, 12, 3, 18, 5, 20"

Your job: Remember the numbers you hear, identify which are even (12, 18, 20), and count them = 3

💡 Tip: Write down numbers as you hear them!

🔊 Test Your Audio:

Answer: There are 2 odd numbers (3, 7)
